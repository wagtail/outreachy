## Introduction

My name is Temidayo Azeez. I am a Backend Developer from Nigeria. I am glad to return as a contributor to the Wagtail project. I first contributed to the Wagtail project in 2022, and it was a great experience that helped me learned a lot within a short time. I must say the same for this year, as I have learned quite a lot about Wagtail and Django than I probably would working on a personal project. 

Open source programmes like Wagtail is the perfect environment for any growing engineer looking to nail a career in software engineering. One, you get to accumulate concrete knowledge for free by getting to work with mentors who are always ready to offer assistance. Second, working on large open source projects helps to understand the task of maintaining large codebases where every action must be carefully designed and executed to ensure code reliabiity and a thriving community. Third, it mirrors a real work environment allowing you to absorb all the necessary skills needed to succeed in a software engineering career. 

Aside Wagtail, I have also contributed to other open source projects such as MindsDB and Jupyterhub. For this year Outreachy programme, I am interested in the `Adopt generic class based views across the Wagtail admin` project. Python/Django is my main stack as a Backend Developer, and this fact got me excited about the project. The journey has been exciting so far. Looking forward to more wins in knowledge, experience and expertise.

## Research

- [Wagtail GitHub Repository](https://github.com/wagtail/wagtail) - This has constituted the bulk of my research in understanding the Wagtail machinery.
- [Wagtail Documentation](https://docs.wagtail.io/en/stable/) - The Wagtail official documentation have been very useful in investigating and reproducing unconfirmed bugs. They have also been useful in helping me better understanding Wagtail code base.
- [Django Documentation](https://docs.djangoproject.com/en/4.2/) - The Django documentation has also come very useful in understanding recent development in Django, which provides a good guide in submitting contributions to Wagtail, and also in keeping the Wagtail machinery in tandem with Django latest updates.
